# 🍳 Meena's Recipe Planner

A professional **React + Vite Recipe Planner Application** that allows users to explore recipes, filter categories dynamically, and save favorite meals using persistent storage.

🔗 **Live Demo:**
https://meena-s-recipe-planner.vercel.app/

---

## ✨ Features

* 🔍 **Live Search Filtering** — Instantly filter recipes by title
* 📂 **Dynamic Category Filtering** — Categories generated automatically from API data
* ❤️ **Favorites System** — Save favorite recipes with live navbar counter
* 💾 **Persistent Storage** — Favorites saved using LocalStorage
* 📋 **Recipe Modal View** — Displays ingredients + instructions interactively
* 🌐 **API Integration** — Recipes fetched from TheMealDB public API
* 📱 **Responsive Layout** — Mobile, tablet, and desktop optimized UI
* ⚡ **Optimized React Hooks Usage** — Clean state management with useState + useEffect

## 🚀 Installation & Setup

Clone repository

```
git clone https://github.com/YOUR_USERNAME/meena-s-recipe-planner.git
```

Install dependencies

```
npm install
```

Run development server

```
npm run dev
```
